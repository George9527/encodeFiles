使用说明：
---
1. 将你要加密的文件复制到Material文件夹下，如果没有Material可以自己新建
2. 双击ClickToEncodeFiles.bat文件，加密Material文件夹下的文件
3. 打开encodeFiles文件夹，其中就是加密后的文件
	
注意：
---
1. 加密的文件名不能有空格
2. 每次会输出加密的结果注意查看。
